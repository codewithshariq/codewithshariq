### Senior Full Stack Engineer

I build full-stack products across React, Next.js, Node.js, TypeScript, payments, real-time systems, and AI-powered workflows. I enjoy turning ambiguous business problems into shipped software: clean interfaces, reliable backends, measurable performance gains, and products people can actually use.

Recently, I have been working as a senior engineer and lead/sole engineer on products involving subscription payments, partner payouts, voice AI agents, analytics dashboards, RAG-based automation, pricing intelligence, and multi-tenant commerce platforms.

### Recent Highlights

- Led **GolfLoot**, a subscription-based prize-draw platform, end-to-end as the sole engineer using Next.js, shadcn/ui, Tailwind CSS, Supabase, and Stripe. Since launch, the platform reached **410 active subscribers**, **A$45K revenue**, and a projected **A$95K ARR**.
- Built a production **voice AI platform** with Vapi, ElevenLabs, Twilio, Next.js, TypeScript, analytics dashboards, prompt/agent controls, and AWS Lambda deployments managed through Terraform.
- Developed an AI email automation prototype using **LangChain, RAG, and vector databases** to learn from inbox history and generate context-aware email drafts.
- Led a 4-engineer team on **Honeycomb Credit**, improving page loads from **1.5s to under 0.5s**, supporting real-time investment totals, Stripe payments, KYC, and a redesigned investment flow that helped drive **2.2x investments** during a campaign event.
- Improved **Cafe Zupas** frontend performance and accessibility, raising accessibility scores from **60 to 90+**, reducing First Contentful Paint from **2.6s to 1.1s**, eliminating redundant API calls with React Query, and cutting bundle size by **17%**.
